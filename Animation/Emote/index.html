🎭 Idle 애니메이션 루프 정책

  • 1. Idle(1)은 항상 기본 루프로 사용 (Idle이 1개만 있으면 해당 Idle만 루프)
  • 2. Idle(2)~Idle(3)은 Idle(1) 이후 랜덤 IdleVariation으로 취급
  • 3. 정지 상태 진입 → Idle(1) 즉시 루프 재생
  • 4. 기본 Idle 루프가 최소 10초 유지 후 → Idle(2)~(3) 중 1회 블렌딩 재생
  • 5. IdleVariation 종료 → Idle(1) 기본 루프 복귀, 반복
