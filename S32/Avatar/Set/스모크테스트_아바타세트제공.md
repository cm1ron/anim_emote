# 아바타 세트 제공 스모크테스트

---

## 테스트 플로우 요약

- **Flow 1 (유저)**: Marketplace Avatars 탭 진입 → 세트 탐색·상세 확인 → 세트 장착(프리뷰) → Save(구매/저장) → 구매내역 확인
- **Flow 2 (운영→유저)**: 유저 Web 쇼케이스 코디 게시 → Hiker에서 운영팀 세트 등록·순서 조정 → Save 반영 → 큐레이터 알림 수신 → Avatars 탭 노출 확인
