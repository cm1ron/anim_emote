## S-1. 번들 팩 구매 → 장착 → 인게임 확인

**전제 조건**
- 미구매 계정 (BLUC 700 이상 보유)
- 아무 번들도 장착되지 않은 상태

| 단계 | 행동                                              | 기대 결과                                                       | logcat 확인                                 |
| ---- | ------------------------------------------------- | --------------------------------------------------------------- | ------------------------------------------- |
| 1    | 마켓플레이스 진입 → `Animations` 탭 선택          | Bundle / Emotes 서브탭 노출                                     | `categoryId: BundleAnimation` 요청 발생     |
| 2    | Skate Pack 선택 (100 Bluc)                        | 가격·썸네일·포함 6종 애니 목록 노출, **구매 버튼** 표시         | `HasEquipableItems: false`                  |
| 3    | 구매 버튼 탭 → 구매 완료                          | "구매 완료" 토스트 노출, 버튼이 **장착 버튼**으로 변경          | `HasEquipableItems: true` (anim:41~46 전환) |
| 4    | 장착 버튼 탭                                      | 6종 슬롯 모두 Skate 썸네일로 교체, Bundle 슬롯 **Skate로 표시** | `A2U_EQUIP_ANIMATION_ITEM` 6회 발생         |
| 5    | `.pak` 다운로드 완료 대기                         | 에셋 다운로드 진행 표시 후 완료                                 | `LogIoDispatcher: Mounting container` 성공  |
| 6    | 커스터마이즈 저장 → 인게임 진입                   | 캐릭터가 **Skate Idle** 포즈로 서있음                           | -                                           |
| 7    | 인게임에서 걷기 / 달리기 / 점프 / 낙하 / 전력질주 | 각각 Skate Walk / Run / Jump / Fall / Sprint 모션 재생          | -                                           |
