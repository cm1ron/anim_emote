📡 무선 연결 방법

방법 1: USB 자동 전환 (권장)

  1. USB로 연결
  2. 앱이 무선 모드(5555) 자동 전환
  3. USB 분리해도 무선 유지

방법 2: 수동 페어링

  1. 설정 → 개발자 옵션 → 무선 디버깅 ON
  2. "페어링 코드로 기기 페어링" 터치
  3. "무선 연결" → IP/코드 입력 → 페어링 → 연결
🤖 Gemini API 키 등록
  1. Google AI Studio 접속
  2. Google 로그인 → "Create API Key" 클릭
  3. 키 복사 → 모니터링 탭 AI ⚙ 버튼 → 키 입력

* 무료: 분당 15회 / 일 1,500회
