✅ TC-1. 템플릿 선택 & 방식 선택 (P0)
  • 1-1 Hub 진입 → AIT 5종(Head/Tshirt/Pants/Gloves/Boots) 템플릿 노출 확인
  • 1-2 각 템플릿에 NEW 뱃지 + AI 아이콘 표시
  • 1-3 (구) 템플릿 Hub에서 hide 확인 (신규 5종 외 미노출)
  • 1-4 (구) 템플릿으로 제작된 판매 중 아이템 → Marketplace에서 유지 확인
  • 1-5 템플릿 클릭 → "Choose How to Create Your Item" 모달 노출
  • 1-6 모달: AI Generate / Upload Image 2개 옵션 노출
  • 1-7 아무것도 미선택 시 → Next 버튼 비활성화 (회색)
  • 1-8 AI Generate 선택 시 → Next 버튼 활성화 (흰색)
  • 1-9 Upload Image 선택 → 기존 수작업 업로드 플로우 100% 동일 진입
  • 1-10 Upload Image: UV 템플릿 다운로드, 예제 이미지 다운로드, 좌측 3D 프리뷰 정상 동작
  • 1-11 모달 X 버튼 → 모달 닫힘, 템플릿 선택 화면 복귀
✅ TC-2. AI Generate — 프롬프트 입력 (P0)
  • 2-1 AI Generate 진입 → Step 2 "AI Generate" 스텝 표시 확인
  • 2-2 좌측: 선택한 파츠의 3D 프리뷰 정상 표시
  • 2-3 우측: "AI-Powered Creation" 패널 노출
  • 2-4 Prompt placeholder: "Describe the style you want..."
  • 2-5 Prompt 최대 500자 제한 → 500자 초과 입력 불가
  • 2-6 글자수 카운터 정상 표시 (예: 81/500)
  • 2-7 Style References 3종 노출: Anime High School / KPOP Star / Gyaru
  • 2-8 Style Reference 클릭 → 예제 프롬프트 자동 입력 + 3D 프리뷰에 예제 텍스처 적용
  • 2-9 Style Reference 클릭 → Generate 버튼 활성화
  • 2-10 Generate 버튼에 잔여 횟수 표시 (예: 5 left)
  • 2-11 이탈 경고 배너 상시 노출: "Leaving this page or refreshing will cancel the generation..."
  • 2-12 하단 Back / Next 버튼 정상 동작
  • 2-13 빈 프롬프트 상태에서 Generate 버튼 동작 확인 (비활성화 or 에러 처리)
✅ TC-3. AI Generate — 생성 & 로딩 (P0)
  • 3-1 Generate 클릭 → 로딩 스피너 + "Generating the image may take up to 1 minute" 메시지
  • 3-2 Generate 클릭 시 잔여 횟수 즉시 1 차감 (예: 5→4)
  • 3-3 생성 중 프롬프트 수정 불가
  • 3-4 생성 중 Style References 클릭 불가
  • 3-5 생성 중 Generate 버튼 비활성화 (중복 생성 방지)
  • 3-6 생성 중 Back 버튼 클릭 → 페이지 벗어남 경고 모달 노출
  • 3-7 생성 중 브라우저 X 버튼 → 시스템 경고 모달 노출
  • 3-8 생성 완료 → 결과 텍스처가 좌측 3D 프리뷰에 자동 적용
✅ TC-4. AI Generate — 에러 처리 (P0)
  • 4-1 생성 실패 → "Generation failed. Please try again." + Retry 버튼
  • 4-2 생성 실패 시 횟수 미차감 확인
  • 4-3 타임아웃 → "Taking longer than expected. Please try again." + Retry 버튼
  • 4-4 타임아웃 시 횟수 미차감 확인
  • 4-5 사용자 취소 → "Cancel generation and exit?" + Cancel and Exit (빨간 버튼)
  • 4-6 사용자 취소 시 횟수 차감됨 확인
  • 4-7 Retry 버튼 클릭 → 에러 모달 닫히고 프롬프트 입력 상태 복귀
✅ TC-5. AI Generate — 결과 확인 (P0)
  • 5-1 생성 완료 → "Your Results" 영역에 UV 텍스처 이미지 표시
  • 5-2 결과 이미지가 좌측 3D 프리뷰에 자동 적용
  • 5-3 프롬프트 영역에 현재 결과물 생성 프롬프트 유지
  • 5-4 프롬프트 수정 후 재생성 가능
  • 5-5 결과 2개 이상 시 → < > 화살표로 이전 결과 탐색 가능
  • 5-6 이전 결과 선택 시 → 해당 프롬프트 & 3D 프리뷰 동기화
  • 5-7 2번째 생성 결과 시 툴팁: "You can go back to your previous design."
  • 5-8 결과 카운터 정상 표시 (예: 1/1, 1/2, 3/5)
  • 5-9 Next 버튼 빨간색으로 활성화 (결과 있을 때)
  • 5-10 Next 클릭 → Step 3 (Provide Information)으로 정상 진행
✅ TC-6. 생성 횟수 관리 (P0)
  • 6-1 초기 생성 가능 횟수 5회 확인
  • 6-2 매 Generate 클릭 시 1회씩 차감 (5→4→3→2→1→0)
  • 6-3 잔여 0회 → Generate 버튼 비활성화 (disable)
  • 6-4 잔여 0회 → 기존 생성 결과 중 선택은 가능
  • 6-5 매일 UTC 0시에 최대 5회까지 충전 확인
  • 6-6 남은 횟수 이월 확인 (예: 3회 남은 상태 → UTC 0시 후 5회)
  • 6-7 Dev/Staging 환경 → 무제한 사용 가능 확인
✅ TC-7. 판매 & 관리 (P0)
  • 7-1 Step 3: Name / Category / Sale (On/Off) / Price (min 50 BLUC) / Tag (최대 5개) 입력 가능
  • 7-2 Step 4: Complete → 심사 프로세스 기존과 동일
  • 7-3 My Contents에서 AI 생성 아이템 노출
  • 7-4 AI 생성 아이템 도안 썸네일에 AI 아이콘 표시
  • 7-5 AI 생성 아이템 메타데이터 수정 시 → 도안(template) 탭 제거 (수정 불가)
  • 7-6 App Marketplace에서 AI 생성 아이템 정상 판매
